# 活字·复生

怀素《大草千字文》AI 集字海报生成器。用户输入几个字，系统用原帖透明字模集成一张可下载、适合朋友圈传播的草书海报，并可调用 `mimo-v2.5` 生成书法讲解文案。

## 功能

- 草书海报：输入 2-24 个汉字，自动竖排成方图或竖版海报，Canvas 渲染高清 PNG。
- 朋友圈传播：内置宣纸、墨拓、朱砂三种视觉风格，一键下载。
- AI 讲解：服务端代理调用 `mimo-v2.5`，生成适合汇报/分享的书法文案。
- 字源证明：每个字显示来自哪一页、哪个坐标，用于评审证明真实数据来源。
- 缺字策略：第一版用离线 AI 补字占位标注；后续可接入字体生成模型批量补全。

## 数据来源

- 母本：唐怀素《大草千字文》，余子俊于明成化六年摹刻，明刊本。
- 本仓库只保留网页可用的渲染页图与坐标元数据；原始上传 PDF 不纳入正式发布说明。

## 本地预览

纯前端可直接打开 `index.html`。如需 AI 讲解，先构建 `dist/`，再设置环境变量并启动：

```bash
MIMO_API_KEY=... MIMO_BASE_URL=https://token-plan-cn.xiaomimimo.com/anthropic MIMO_MODEL=mimo-v2.5 python3 server.py
```
